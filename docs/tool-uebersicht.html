⚡ Grösste Einsparpotenziale

💡 Quentn ersetzen → n8n + Brevo/Mailgun
Quentn wird nur für Double-Opt-In und einfache E-Mail-Sequenzen genutzt. Das kann n8n mit einem günstigen Transaktions-E-Mail-Dienst (Brevo Free: 300 Mails/Tag, Mailgun: 1'000 Mails/Monat kostenlos) selbst übernehmen. Ersparnis: ~€35-50/Monat.
💡 Pipedrive ersetzen → n8n DataTables + Notion
Pipedrive wird hauptsächlich als Kontakt-Datenbank genutzt (kein Sales-Pipeline). n8n DataTables können Kontakte direkt speichern, Notion als kostenloses CRM dienen. Ersparnis: ~€15-25/Monat.
💡 Gamma.app ersetzen → jsPDF direkt in n8n
Briefing-PDFs werden aktuell über Gamma.app generiert (externes API + Polling). Mit der jsPDF-Bibliothek direkt in einem n8n Code-Node könnten PDFs sofort generiert werden — schneller und ohne externes Abo. Ersparnis: Credits + Latenz.
💡 Short.io → eigene Kurzlinks via n8n + 404.html
Das bestehende 404.html-Routing (für WhatsApp-Funnel) kann auch für Herzkraft-Links genutzt werden. Kurzlinks werden in n8n DataTable gespeichert, 404.html macht den Redirect. Kein externer Dienst nötig. Ersparnis: Short.io-Abo komplett.

5 · Optimierungsvorschläge — Weniger Abos, mehr Leistung

🔴 Priorität 1 — Grösste Kostenersparnis

Quentn → n8n + Brevo (oder Mailgun) ~€40/Monat sparen
Quentn wird nur für Formular-Empfang und Double-Opt-In genutzt. Beides kann n8n selbst:
  • Formular: Bereits per n8n-Webhook empfangen (herz-analyse.html sendet schon an n8n)
  • Double-Opt-In: n8n generiert Token → speichert in DataTable → sendet Bestätigungsmail via Brevo/Mailgun → Klick bestätigt
  • E-Mail-Sequenzen: n8n Schedule-Trigger mit Wartezeiten statt Quentn-Funnels
Aufwand: ~4-6 Stunden · Risiko: Mittel (E-Mail-Zustellbarkeit testen)
Pipedrive → n8n DataTables + Notion CRM ~€20/Monat sparen
Pipedrive wird als Kontakt-Datenbank genutzt — keine echte Sales-Pipeline im Einsatz.
  • Kontakte: n8n DataTable mit Name, E-Mail, Telefon, Notizen, Tags
  • Suche: n8n Code-Node mit Filter (Kaskade wie bisher: E-Mail → Name → Telefon)
  • Notizen: Direkt in DataTable als JSON-Feld
  • Alternative: Notion als kostenloses CRM-Frontend (Datenbank-Ansicht)
Aufwand: ~6-10 Stunden · Risiko: Hoch (viele Workflows betroffen — schrittweise migrieren)

🟡 Priorität 2 — Vereinfachung

Short.io → Eigenes Kurzlink-System (404.html) ~€10/Monat sparen
Das 404.html-Routing existiert bereits für den WhatsApp-Funnel:
  • Aktuell: go.myimpactflow.ch/CODE → 404.html → Redirect mit ?wa= und ?src=
  • Erweiterung: n8n DataTable als Link-Datenbank, 404.html liest Code → sucht Ziel-URL → leitet weiter
  • Klick-Tracking: 404.html sendet sendBeacon an n8n vor dem Redirect
  • Vorteil: Volle Kontrolle, keine API-Limits, keine Drittanbieter-Abhängigkeit
Aufwand: ~3-4 Stunden · Risiko: Niedrig
Gamma.app → jsPDF in n8n Code-Node Credits sparen
Gamma wird für Briefing-PDFs und Health-Reports genutzt. Die Polling-Wartezeit (bis zu 3 Min) ist ein Bottleneck.
  • Alternative: jsPDF/PDFKit in einem n8n Code-Node — sofortige PDF-Generierung
  • Design: HTML → PDF mit Puppeteer (n8n Community Node) für hübsche PDFs
  • Vorteil: Sofortige Erstellung (kein Polling), ImpactFlow-Branding konsistent
Aufwand: ~4-6 Stunden · Risiko: Niedrig

🟢 Priorität 3 — Zukunfts-Optimierung

OpenRouter → Direkt-API (OpenAI / Anthropic) Eventuell günstiger
OpenRouter nimmt einen kleinen Aufschlag (~5-15%) auf die API-Preise. Bei steigendem Volumen könnte ein direkter API-Zugang günstiger sein.
  • Vorteil OpenRouter: Multi-Modell-Zugang (GPT-4o, Claude, Gemini) über einen Key
  • Vorteil Direkt: ~10% günstiger, weniger Latenz, kein Mittelsmann
  • Empfehlung: Erst ab ~€50/Monat OpenRouter-Kosten wechseln
Brave Search → SearXNG (Self-Hosted) Langfristig
Bei Überschreitung des Free-Tiers könnte eine eigene SearXNG-Instanz (Open Source) unbegrenzte Suchen ermöglichen.
